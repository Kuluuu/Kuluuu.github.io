环状序列(Circular Sequence,ACM/ICPC Seoul 2004,UVa1584)

发表于 2019-09-29 | 分类于 算法竞赛入门经典

题目描述

长度为 n 的环状串有 n 种表示法，分别为从某个位置开始顺时针得到的。在环状串的所有表示法中，字典序最小的称为“最小表示”。
输入一个长度为 n 的环状DNA串（只包含A ，C ，G ，T 这四种字符）的一种表示法，你的任务是输出该环状串的最小表示。

输入

2
CGAGTCAGCT
CTCC

输出

AGCTCGAGTC
CCCT

原题（PDF）

题目分析

字典序最小，可认为要求第i位的字符为当前可选的第i位的字符中最小的。

案例分析

例如案例中 CGAGTCAGCT
第一位最小字符为：A ，因此从两个 A 可能的字符开始组成字符串
第二位最小字符比较：G与G
第三位最小字符比较：T与C ， 因此可知AGC当前字符为“最小表示”

解题标程

错题总结

  • 1、用字符串字典顺序的总和数值判断是否为最小不合理，因为无法判断当前位置是否为此刻最小
  • 2、采用结构化编程，将比较过程写成自定义比较函数，遍历所有可能的字符串起始位置进行比较
