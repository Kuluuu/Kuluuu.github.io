假定一个解，再去判断这个解是否可行，再不断缩小范围

 

例题：

题目描述

为了购买网络线缆，评委会联系了当地的网络解决方案提供商，要求为他们出售一定数量的等长度的线缆。评审委员会希望电报越长越好，让参赛者坐得越远越好。

公司的电缆主管被派去执行这项任务。他知道库存中每根电缆的长度，最多是一厘米，他可以精确地切割每根电缆，并被告知必须切割的部分的长度。然而，这一次，长度不知道和电缆主人完全困惑。

你要帮助电缆主，通过编写一个程序，将确定最大可能长度的电缆片可以从电缆切断的股票，以获得指定的数量。

输入描述

输入文件的第一行包含两个用空格分隔的整数N和K。N (1 <= N <= 10000)是库存电缆的数量，K (1 <= K <= 10000)是请求的数量。第一行后面是N行，每行有一个数字，表示库存中每根电缆的长度(以米为单位)。所有的电缆都至少1米长，最多100公里长。输入文件中的所有长度都以厘米精度写入，小数点后正好有两位数字。

输出描述

写入输出文件的最大长度(米)，电缆主可以削减从电缆的股票，以获得要求的件数。数字的书写精度必须为厘米，小数点后必须有两位数字。

如果无法将所请求的片段数量(每个片段至少有1厘米长)分割，则输出文件必须包含单个数字“0.00”(不带引号)。

输入

4 11
8.02
7.43
4.57
5.39

输出

2.00

三、最大化最小值

通过二分法也可以很好的解决最大化最小值的问题

 

例题：

题目描述

农民约翰建了一个新的长谷仓，有N个摊位。摊位位于x1、…，xN (0 <= xi <= 1,000,000,000)。

他的C (2 <= C <= N)头奶牛不喜欢这种谷仓布局，一旦被关进牛栏，它们就会变得咄咄逼人。为了防止奶牛互相伤害，FJ想要将奶牛分配到牛棚中，使它们之间的最小距离尽可能的大。最大最小距离是多少?

输入描述

  • 第1行：两个用空格分隔的整数：N和C

  • 第 2..N+1行： 第i+1行包含一个整数摊位位置，xi

输出描述

  • 第1行：一个整数：最大最小距离

输入

5 3
1
2
8
4
9

输出

3

提示

FJ可以把他的3头奶牛放在1号、4号和8号位置的牛栏里，这样就能保证3只奶牛之间的最小距离。

庞大的输入数据，建议使用scanf。

关于二分精度的总结

1、可以通过控制二分循环来控制二分精度，可以将终止条件设置为100次的循环，那么精度就可以达到10-30的精度范围，同样也可以把终止条件设为 (ub - lb) > EPS 这样，但是注意，如果EPS取得太小，就有可能因为浮点小数精度的原因导致陷入死循环
